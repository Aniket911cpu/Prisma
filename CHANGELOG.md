# Changelog

All notable changes to the "Prisma" extension will be documented in this file.

## [1.0.0] - 2025-12-28

### Added
- **Global EyeDropper**: Pick colors from any web page.
- **Glassmorphism UI**: A stunning, modern dark-mode interface.
- **Smart History**: Saves last 50 picked colors.
- **Magic Palettes**: Auto-generates Monochromatic, Analogous, and Complementary schemes.
- **Contrast Checker**: Instant WCAG accessibility checking.
- **Options Page**: Settings for Auto-Copy and History management.
- **Growth Engine**: Smart "Rate Us" nudge and Social Sharing.
