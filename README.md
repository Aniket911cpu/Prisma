# Prisma: The Ultimate Color Workflow Studio

**Prisma** is a refreshing, production-grade color picker extension for Chrome, designed for designers, developers, and color enthusiasts. It replaces outdated tools like ColorZilla with a modern, glassmorphic interface and a suite of powerful features.

## Why Prisma?

| Feature | Prisma | ColorZilla |
| :--- | :--- | :--- |
| **Interface** | Stunning Glassmorphism (Dark Mode) | Outdated Windows 95 style |
| **Experience** | Smooth animations, confetti, instant feedback | Clunky, static |
| **Palettes** | Auto-generates Mono, Analog, Comp schemes | Basic history only |
| **Accessibility**| Built-in WCAG AA/AAA Contrast Checker | None |
| **Privacy** | 100% Local (no analytics) | Often tracks usage |

## Installation

1. Clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select the `Prisma` folder.

## Permissions Explained

- `activeTab`: To pick colors from the current tab.
- `storage`: To save your color history and settings locally.
- `scripting`: To support color picking mechanics.

## Privacy Policy

Prisma is offline-first. We do not collect, store, or transmit your personal data or browsing history. Everything stays on your machine.
